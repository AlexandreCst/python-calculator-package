# Python calculator OOP package

## Description
CLI calculator in Python based on POO principles and packaged. The script ask 
numbers and an operator among [+, -, *, /, **, %, !, sqrt] to make some 
scientific calculations.

## Features
- Package with basic (class Calculator), scientific (subclass ScientificCalculator) and helpers (utils functions)
- Class calculator with methods (addition, subtraction, multiplication, 
division, show history and clear history)
- Subclass scientific with methods (power, factorial, modulo and square root)
- Executable program main script with basic and scientific arithmetic operations (addition, subtraction, multiplication, division)
- Save history in txt file and possibility to load history from a txt file 
- Input validation
- Calculation history (viewable on demand)
- Exit anytime with 'q' or 'quit' command
- Formatted output with 2 decimal places

## Installation
Clone the repository:
```bash
git https://github.com/AlexandreCst/python-calculator-package.git
cd python-calculator-package
```

No additional dependencies required (uses Python standard library only).

## How to Use
Run the program:
```bash
python main.py
```

Follow the prompts:
- Enter your first number
- Enter your second number
- Choose an operation (+, -, *, /, **, %, !, sqrt)
- View result
- Type 'q' anytime to quit

## Requirements
- Python 3.9 or above
